Run this program first by running FtpServer.java, then FtpClient.java.

Console output will appear in the FtpClient.java run terminal, and typing commands in that end after each "Command:" prompt will cause the program to complete that command.

Accepted commands are

LS
PWD
GET
PUT
QUIT

All other commands will result in an exception.

Entering the names of files that do not exist may result in an exception.
